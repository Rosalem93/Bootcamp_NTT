		<h2>O que é o HTML</h2>
		<p>HTML é a linguagem de marcação padrão para páginas da Web.</p>
		<p>Com HTML você pode criar seu próprio site.</p>

		<h2>Atributos do HTML</h2>
		<h3>Os atributos HTML fornecem informações adicionais sobre elementos HTML</h3>
		<p>Todos os elementos HTML podem ter atributos</p>
		<p>Os atributos fornecem informações adicionais sobre os elementos</p>
		<p>Os atributos são sempre especificados na tag inicial</p>
		<p>Os atributos geralmente vêm em pares nome/valor como: name="value"</p>

		<h2>Listas HTML</h2>
		<p><strong>As listas HTML permitem que os desenvolvedores da Web agrupem um conjunto de itens relacionados em listas.</strong></p>
		<h3>Lista HTML não ordenada</h3>
		<p>Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão</p>
		<h4>Exemplo:</h4>
			<ul>
				<li>HTML</li>
				<li>CSS</li>
				<li>JavaScript</li>
			</ul>
		<h3>Lista HTML ordenada</h3>
		<p><strong>Os itens da lista serão marcados com números por padrão</strong></p>
		<h4>Exemplo</h4>
			<ol>
				<li>HTML</li>
				<li>CSS</li>
				<li>JavaScript</li>
			</ol>
